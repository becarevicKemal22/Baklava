Map of all keywords in the language and their corresponding Token\+Type. 

Used by the \doxylink{class_lexer}{Lexer} when an identifier is found to check if it is a keyword. Changing this map enables easy modification of the language\textquotesingle{}s keywords without affecting any other code (except tests).
