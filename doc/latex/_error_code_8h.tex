{\footnotesize\ttfamily enum \mbox{\hyperlink{_error_code_8h_a59e56af19e754a6aa26a612ebf91d05f}{Error\+Code}}}



Standardized error codes for the interpreter that should be used by the entire project. 

Every single possible error is included in the enum, and no error reporting should be done without using the correct code. 1xx are general errors. 2xx are lexical errors. 3xx are parsing errors. 4xx are interpreter runtime errors. \begin{DoxySeeAlso}{See also}
\doxylink{_error_messages_8h}{Error\+Messages.\+h} for the error messages corresponding to each code. 
\end{DoxySeeAlso}
