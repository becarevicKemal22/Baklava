Detailed Description

Base class used for printing error messages to the console.

Should be passed as a pointer to the lexer, parser and interpreter and is used to print all error messages. All error messages should go through the error printer or a class derived from it.

Mainly used for errors that occur before the lexer is initialized, most likely 1xx errors.
