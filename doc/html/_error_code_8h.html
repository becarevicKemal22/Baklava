enum ErrorCode

Standardized error codes for the interpreter that should be used by the entire project.

Every single possible error is included in the enum, and no error reporting should be done without using the correct code. 1xx are general errors. 2xx are lexical errors. 3xx are parsing errors. 4xx are interpreter runtime errors.

See also
ErrorMessages.h for the error messages corresponding to each code.
